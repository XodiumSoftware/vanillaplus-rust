# Commands & Permissions

All permission nodes follow the pattern `pumpkinplus.command.<name>`.

## Locator

| Command                        | Aliases | Permission                    | Default  | Description                 |
|--------------------------------|---------|-------------------------------|----------|-----------------------------|
| `/locator <color\|hex\|reset>` | `lc`    | `pumpkinplus:command.locator` | Everyone | Set your locator bar colour |

> **Note:** The locator command is registered but not yet implemented — the underlying Pumpkin API is not ready.

All permissions default to **Everyone**.
